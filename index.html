Skill Level Indicator

Canvas-based skill indicator

Download .zip Download .tar.gz View on GitHub

Welcome to Skill Level Indicator's GitHub Page.

Skill Level Indicator is a javascript function that helps you create nice and highly customizable rounded skill indicators. It's usage is very simple. It only requires a browser that supports the html5 canvas element.

See it in action: http://sandbox.apovtx.eu/SkillLevelIndicator/

Usage

After including the script in your page you only have to just call it with 1 mandatory argument and some optional changes you may want.

To include it in your page you just need to have:

<script src="js/Skill.js"></script>
<canvas id="myCanvas" width="960" height="200"></canvas>

and to create a skill indicator:

<script type="text/javascript">
var canvas = document.getElementById("myCanvas");
var context = canvas.getContext("2d");

SkillIndicator(context, 
{
X: canvas.width / 2,
Y: canvas.height / 2,
radius: 70,
skill: 65,
name: 'CSS'
});
</script>

And that's all!

Options

You can customize the output with many different options:

  • color : To change the background color of the circle
  • borderColor : TO change hte skill indicator color
  • X : The position of the circle in the x-axis
  • Y : The position of the circle in the y-axis
  • level : The % level of your skill
  • name : The name of the skill
  • radius : The radius of the circle
  • lineWidth : The width of the indicator
  • fontSize : The font size of the skill name
  • fontColor : The color of the text
